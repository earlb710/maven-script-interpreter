4-Quadrant Board Game - Complete Board with Corners

Center cell (C0,0) + 4 arms (8 cells each) + 4 shared corner cells (S) + 4 borders (16 cells each) + 4 corner cells (C)
Total: 1 center + 32 arm cells + 4 shared corners + 64 border cells + 4 corner cells = 105 unique cells

Cell Structure

  • Center Cell (Gold): C(0,0) - Starting position
  • Arm Cells (Blue): 32 cells (8 per direction: N1-N8, E1-E8, S1-S8, W1-W8)
  • Shared Corner Cells (Orange): 4 cells (S-N, S-E, S-S, S-W)
  • Border Cells (Green): 64 cells (16 per side: B1-B64)
  • Board Corner Cells (Purple): 4 cells (C-NW, C-NE, C-SE, C-SW)

Board Statistics

  • Center: 1 cell (C0,0)
  • Arms: 32 cells (4 directions × 8 cells each)
  • Shared Corners: 4 cells (at the end of each arm)
  • Borders: 64 cells (4 borders × 16 cells each)
  • Board Corners: 4 cells (at the 4 corners of the square board)
  • Total: 1 + 32 + 4 + 64 + 4 = 105 unique cells

Border Structure (per side)

  • Each border: 1 corner + 8 cells + 1 shared corner + 8 cells + 1 corner = 19 positions
  • Top border: C-NW + B1-B8 (8) + S-N (shared) + B9-B16 (8) + C-NE
  • Right border: C-NE + B17-B24 (8) + S-E (shared) + B25-B32 (8) + C-SE
  • Bottom border: C-SE + B33-B40 (8) + S-S (shared) + B41-B48 (8) + C-SW
  • Left border: C-SW + B49-B56 (8) + S-W (shared) + B57-B64 (8) + C-NW

Gameplay Notes

  • Players start at C(0,0) in the center
  • Move along one of the 4 arms (North, East, South, or West)
  • Reach shared corners (S-N, S-E, S-S, S-W) to access the border paths
  • Border paths form the outer perimeter of the game board
  • Board corners (C-NW, C-NE, C-SE, C-SW) mark the 4 corners of the square board
