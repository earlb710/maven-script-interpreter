# Board Game Project

This is an EBS (Earl Bosch Script) project for developing a board game application.

## Project Structure

- `project.json` - Project configuration file
- `main.ebs` - Main entry point script
- `.copilot-instructions.md` - AI assistant instructions

The `project.json` file contains:
- Project name and description
- Version information
- CSS files for UI styling
- Main script entry point
- Custom settings
